Getting started

In this section you'll learn how to get started with proteus-charts.

Usage

Include the dependencies

The first step is to use the library is to include the JS library in your HTML page. For the moment we have not hosted it in any cdn server. It is only available on rawgit one:

 <script src='https://rawgit.com/proteus-h2020/proteus-charts/development/dist/proteus-charts.js'></script>

If you don't like cdns you can download the library and include it as a local script:

 <script src='./dist/proteus-charts.min.js'></script>

Start graphing!

Once you have include the library, you are ready to start graphing. Take a look into graphs section to see the full list of available graphs and how they works.

Run local examples

Previously to run the examples you need to install git, or to manually download and unzip proteus-charts. You also will need to install npm, in order to local install the project dependencies 

git clone https://github.com/proteus-h2020/proteus-charts && cd proteus-charts
npm install
npm run serve

Now you can browse on http://localhost:8888 and see the full list of working examples
